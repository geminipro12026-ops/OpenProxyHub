# OpenProxyHub Project Rules

1. Mobile First
The Android application is the primary interface.

2. Plugin Based
No VPN panel is hardcoded into the core.

3. Open Source
The project will remain open source.

4. Extensible
Any future VPN panel can be added using a plugin.

5. Clean Architecture
Every component should be modular.

6. Security First
Sensitive information must always be encrypted.

7. Simplicity
The application should remain easy to use.
